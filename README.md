## What I'm building

**🤖 [ryvAI](https://ryvai.platformnx.com)** — Production AI code reviewer. ~100K lines TypeScript monorepo across 5 packages. 6 specialist agents (security, correctness, blast radius, compliance, performance, style) run in parallel, consolidated by an LLM judge with adversarial self-reflection. Active across 50+ repos on GitHub and Azure DevOps. Cuts 2–4 false positive review comments per PR.

**🧪 [AceCodingLab](https://acecodinglab.com)** — Interview prep for frontend engineers. DSA, real-world UI challenges, system design. 500+ monthly organic users, zero paid acquisition.

**🗺️ [Mapify](https://mapify.vercel.app)** — Node-based roadmap and flowchart editor. Build visual roadmaps, save as PNG or embed anywhere. Built with ReactFlow, Next.js, Firebase.

**📝 [videoNotes](https://video-notes-pi.vercel.app)** — Converts YouTube videos into structured notes, summaries, and blog posts using AI. Next.js + GPT-3 + Editor.js.

**⚙️ [code-execution-environment](https://github.com/theashishmaurya/code-execution-environment)** — Safe, isolated code execution for AI agents and online editors. Written in Go. Powers the autonomous file-edit flow in ryvAI's `@ryv fix` command.

**🧠 [byonn](https://github.com/theashishmaurya/byonn)** — Build your own neural network. TypeScript implementation from scratch, no ML libraries.
